# Entrega 1 - Sistema de Ciber-Defensa Multijugador

## 🎯 Objetivo
Simular un entorno de red donde se implementa un protocolo de aplicación propietario sobre TCP para la gestión de recursos críticos.

## 📁 Archivos en esta carpeta
- `server.c`: Servidor Multihilo.
- `client.py`: Cliente de Usuario.
- `login_server.py`: Lobby Web.
- `protocol_rfc.md`: Especificación oficial del protocolo.
- `demo.sh`: Script de despliegue automático.

## 🛠️ Cómo ejecutar
1. Ejecuta `./demo.sh` para iniciar el entorno completo.
2. Identifícate con `AUTH <nombre>`.
3. ¡Defiende o Ataca!
